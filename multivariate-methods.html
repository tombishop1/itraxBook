Using Itrax Data in R

Chapter 6 Multivariate Methods

Input data often needs extensive preparation if useful results are to be obtained — transformation, dealing with zero values, and NA values. It is worth mentioning some general issues around the robustness of multivariate analysis - special treatment is required for compositional data for most methods; see Aitchison (1986).

At present the preparation of data is handled in itraxR using the internal function itraxR:::multivariate_import(), but at least at present this function is not exported from the package. However it can be accessed by the user using the ::: notation.

data("periodicTable")
CD166_19_xrf %>%
  mutate(ids = 1:dim(CD166_19_xrf)[1]) %>%
  itraxR:::multivariate_import(elementsonly = TRUE, 
                               zeros = "addone", 
                               transform = TRUE) %>%
  glimpse()
